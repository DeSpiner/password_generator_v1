# Password Generator and Brute-Force Simulation

This is a **basic animated password generator** written in Python.  
It generates a random password based on user preferences and simulates a brute-force attack to crack it.

---

## Features

- Animated text output for a nicer console experience
- Customizable password options:
  - Include uppercase letters
  - Include numbers
  - Include special characters
  - Choose password length
- Simulated brute-force attack shows how many attempts it takes and the total time

---

## How to Use

1. Clone or download this repository.
2. Run the script in your terminal:

```bash
python password_generator_basic.py
